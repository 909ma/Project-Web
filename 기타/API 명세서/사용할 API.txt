1. The Movie Database (TMDb):

TMDb는 영화 및 TV 프로그램 정보를 제공하는 대표적인 영화 데이터베이스입니다.
TMDb API를 사용하면 영화의 상세 정보, ** 포스터 **, 배우 정보 등 다양한 데이터를 얻을 수 있습니다.
API 문서: https://www.themoviedb.org/documentation/api

----------------------------------------------------
2. 영화진흥위원회 kofic:

kofic에서 일별 박스오피스, 영화목록, 영화 상세정보, 영화사 목록, 영화인 목록 등을 가져옵니다.
API 문서: https://kobis.or.kr/kobisopenapi/homepg/apiservice/searchServiceInfo.do

----------------------------------------------------
아래는 사용할지 안 할지 모르는 API입니다.
----------------------------------------------------
3. OMDb API:

OMDb는 영화 데이터베이스로부터 영화 정보를 검색하고 가져올 수 있는 API입니다.
API를 사용하여 영화의 제목, 상세 정보, 포스터, 출연 배우 등을 얻을 수 있습니다.
API 문서: http://www.omdbapi.com/

----------------------------------------------------
4. IMDb API:

IMDb는 인터넷 영화 데이터베이스로 유명한 영화 정보 사이트입니다.
IMDb API는 IMDb의 데이터를 활용하여 영화 정보, 평점, 리뷰, 포스터 등을 제공합니다.
API 문서: https://developer.imdb.com/
